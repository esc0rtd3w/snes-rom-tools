GFX27.bin Editing Assistant 

1) Put GFX27.bin in the same folder as this program.
2) Run 3to4.exe, this creates a file named GFX27(4bpp).bin
3) Open GFX27(4bpp).bin in YY-CHR
4) Edit as 4BPP MSX graphics.  Use only the first 8 colors
   of the palette.
5) Run 4to3.exe, this creates a file named GFX27(3bpp).bin that
   can be inserted back into Lunar Magic.
